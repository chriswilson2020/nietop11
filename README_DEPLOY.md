# Niet op 11 — static website

Dutch-first campaign website with English toggle.

This version keeps the broad campaign frame — early advice, self-confirming tracking, noisy-profile children, hidden class mechanisms and talent waste — and adds a fact-assertive section: “Geen quota nodig”. The point is not to defend whether there is a formal quota, but to show that the system produces quota-like sorting through architecture, norming, stable distributions and class-sensitive judgement.

Deploy the folder to any static host: Netlify, Vercel, GitHub Pages, Cloudflare Pages, or normal web hosting.
